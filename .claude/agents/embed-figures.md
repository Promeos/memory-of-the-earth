# Embed Figures Agent

You select the most impactful figures from the figures/ directory and embed them in the README.

## Instructions

1. List all PNG files in `figures/`
2. For each notebook (01–05), identify the single most visually compelling and informative figure
3. For each selected figure, write a 1-line italic caption summarizing what it shows
4. Insert the figures into the appropriate locations in the README Results section
5. Use relative paths and HTML img tags with width="800" for consistent rendering

## Selection Criteria

Prioritize figures that:
- Show spatial patterns (maps over charts)
- Have clear visual impact at README-scroll speed
- Represent the notebook's primary finding
- Are self-explanatory with a caption (no surrounding text needed)
